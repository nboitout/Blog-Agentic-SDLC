## Agentic stack

Kurssi ajattelee software deliveryn uudelleen maailmassa, jossa AI coding agents osallistuvat suoraan planning, coding, testing ja review -vaiheisiin. Kyse ei ole kehittäjien korvaamisesta, vaan systems, constraints ja feedback loops -rakenteiden suunnittelusta luotettavaa ihminen-agentti-yhteistyötä varten.

| Level | Coverage |
|---|---|
| **Agentic SDLC** | Planning, coding, testing and review with AI agents |
| **Agentic engineering** | Loop design, state management and scope boundaries |
| **Harness engineering** | Constraints, verification and continuity |

## Outcomes

- Diagnose why an agentic workflow fails and which harness primitive is missing
- Design a minimal harness for a real codebase
- Coordinate multiple agents across frontend, backend and tests
- Define autonomy levels and human checkpoints aligned with team risk
