## Agentic stack

Цей курс переосмислює software delivery для світу, де AI coding agents беруть участь у planning, coding, testing і review. Йдеться не про заміну розробників, а про дизайн systems, constraints і feedback loops, які роблять співпрацю людини й агента надійною.

| Level | Coverage |
|---|---|
| **Agentic SDLC** | Planning, coding, testing і review з AI agents |
| **Agentic engineering** | Loop design, state management і scope boundaries |
| **Harness engineering** | Constraints, verification і continuity |

## Outcomes

- Діагностувати, чому agentic workflow ламається і якої harness primitive бракує
- Спроєктувати minimal harness для реального codebase
- Координувати кількох агентів над feature, що охоплює frontend, backend і tests
- Визначати autonomy levels і human checkpoints відповідно до ризику команди
