## Agentic stack

Acest curs regândește software delivery pentru o lume în care AI coding agents participă direct la planning, coding, testing și review. Nu este despre înlocuirea dezvoltatorilor, ci despre proiectarea sistemelor, constrângerilor și feedback loops care fac colaborarea om-agent fiabilă.

| Level | Coverage |
|---|---|
| **Agentic SDLC** | Planning, coding, testing și review cu AI agents |
| **Agentic engineering** | Loop design, state management și scope boundaries |
| **Harness engineering** | Constraints, verification și continuity |

## Outcomes

- Diagnosticați de ce un agentic workflow eșuează și ce harness primitive lipsește
- Proiectați un minimal harness pentru un codebase real
- Coordonați mai mulți agenți pe o feature care acoperă frontend, backend și tests
- Definiți autonomy levels și human checkpoints adaptate riscului echipei
