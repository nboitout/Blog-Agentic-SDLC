# Zašto tradicionalni SDLC ne uspijeva s AI agents

> Ovaj tečaj promišlja software delivery za svijet u kojem AI coding agents izravno sudjeluju u planning, coding, testing i review. Nije riječ o zamjeni developera, nego o dizajnu systems, constraints i feedback loops koji čine suradnju čovjek-agent pouzdanom.

## Key concepts

- **Scope**: what the agent may touch and what remains out of bounds.
- **State**: progress recorded in files, not assumed inside the context window.
- **Verification**: automated checks that cannot be bypassed.
- **Continuity**: handoff context sufficient for the next session.

## Business takeaway

The question is not whether the model can generate code. The question is whether the delivery system makes agent work auditable, recoverable and aligned with the team’s risk tolerance.
