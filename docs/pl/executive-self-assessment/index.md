# Samoocena dla kadry zarządzającej

This three-minute executive self-assessment helps leaders understand whether AI is still an individual productivity tool or already part of a governed delivery system.

## Score yourself

Give 1 to 4 points for each area:

1. AI usage in software teams
2. Verification after AI-generated code
3. Workflow redesign around agents
4. Comfort with agent-created pull requests
5. Testing and verification maturity
6. Response to wrong AI output
7. Ownership of human-agent workflows
8. Governance model
9. Value model for AI in delivery
10. Ambition for the next 12 months

## Profiles

- **10-16 Tooling Curious**: AI is mainly individual tooling.
- **17-24 Local Acceleration**: teams see local gains, but governance is fragmented.
- **25-32 Managed Agentic Delivery**: bounded agent workflows exist.
- **33-40 AI-Native Delivery System**: delivery is being redesigned around governed human-agent collaboration.
