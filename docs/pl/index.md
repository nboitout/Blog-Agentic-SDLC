## Agentic stack

Ten kurs redefiniuje software delivery w świecie, w którym AI coding agents uczestniczą bezpośrednio w planning, coding, testing i review. Nie chodzi o zastąpienie developerów, lecz o projektowanie systemów, constraints i feedback loops, które czynią współpracę człowiek-agent niezawodną.

| Level | Coverage |
|---|---|
| **Agentic SDLC** | Planning, coding, testing și review cu AI agents |
| **Agentic engineering** | Loop design, state management și scope boundaries |
| **Harness engineering** | Constraints, verification și continuity |

## Outcomes

- Diagnozować, dlaczego agentic workflow zawodzi i której harness primitive brakuje
- Zaprojektować minimal harness dla realnego codebase
- Koordynować wiele agentów nad feature obejmującą frontend, backend i tests
- Ustalać autonomy levels i human checkpoints zgodnie z ryzykiem zespołu
