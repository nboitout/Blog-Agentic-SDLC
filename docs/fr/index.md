## Qu'est-ce que le SDLC Agentique ? {#quest-ce-que-cest}

Le **SDLC Agentique** est une redéfinition du cycle de développement logiciel autour des agents IA comme participants à part entière. Il ne s'agit pas d'automatiser les développeurs — il s'agit de concevoir les **boucles, contraintes et systèmes de rétroaction** qui permettent aux agents IA et aux ingénieurs humains de collaborer de manière fiable sur des logiciels complexes.

Ce cours synthétise les réflexions les plus avancées du domaine — de la recherche sur le harness engineering d'Anthropic, aux rapports de terrain d'OpenAI sur Codex, en passant par les patterns de production observés dans des équipes livrant avec Claude Code et Cursor.

## Thèmes principaux que vous maîtriserez {#themes}

- **Conception de boucle** — comment structurer les cycles Plan → Code → Test → Revue pour les agents
- **Ingénierie du contexte** — ce que votre agent doit savoir, quand, et dans quel format
- **Primitives de harnais** — gestion d'état, vérification, observabilité, et contrôle de l'entropie
- **Coordination multi-agents** — patterns d'orchestration pour des équipes agentiques parallèles
- **Gouvernance** — quand et comment remettre les humains dans la boucle
